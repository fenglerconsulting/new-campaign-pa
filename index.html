Ablauf
Was passiert, wenn du sagst: "Erstelle Kampagne für CEREC bei Mustermann"
1
Agent — Trello Notes Sync
Trello-Kommentare synchen
Frische Daten aus FenglerKunden, FenglerEditing, FenglerVideografie, FenglerContentplanungKunden und dem Kundenboard holen.
Sucht nach kundenspezifischen Hinweisen: abweichender Radius, Budget, Ad-Copy-Vorgaben, Creative-Präferenzen, Formular-Änderungen.
2
Agent — Rückfrage
Hinweise dem User vorlegen
Gefundene Trello-Hinweise werden zusammengefasst und der User wird gefragt, ob sie noch gelten — mündliche Absprachen haben immer Vorrang.
Fragt auch: mehrumsatz.ai oder Kunde qualifiziert selbst? Das bestimmt Frage 4 + Danke-Seite im Formular.
3
Agent — Creative-Analyse
Creatives analysieren und nach Konvention umbenennen
Bilder werden visuell analysiert (Motiv + Text erkannt). Videos werden aus dem Dateinamen abgeleitet. Alle Creatives bekommen einheitliche Namen — so kann später ausgewertet werden, welcher Creative-Typ (z.B. Fräsmaschinen-Bild vs. Reel) über alle Kunden hinweg die meisten Leads bringt.
4.png → cerec_geraet-fraesmaschine_90min-ohne-abdruck_01.png
Prüft auch: Stadt auf dem Bild korrekt? Format unterstützt?
4
Automatisch — Meta API
Formular erstellen
Instant Form mit Intro-Text, Qualifying Questions, Kontaktdaten und Datenschutz-Link. Variante je nach Vorqualifikation (WhatsApp oder Telefon).
Fragen: Investitionsbereitschaft → Versicherung → Ratenzahlung → WhatsApp/Erreichbarkeit → Name, Telefon, E-Mail
5
Automatisch — Meta API
Creatives hochladen
Bilder und Videos werden zum Werbekonto hochgeladen. Videos über 50 MB automatisch per Chunked Upload. Thumbnail wird automatisch abgerufen.
6
Automatisch — Meta API
Kampagne → AdSet → Creatives → Ads erstellen
Komplette Kampagnenstruktur wird in einem Durchlauf erstellt — inklusive aller Einstellungen, Targeting und Instagram-Verknüpfung.
Ergebnis: Mustermann - CEREC - 05.04.26 mit Ads wie cerec_geraet-fraesmaschine_90min_01 | C1
7
Automatisch — Trello
FenglerKunden-Karte verschieben
Die Kundenkarte wird automatisch in die Spalte „Überprüfen, ob Werbekampagnen laufen" verschoben — mit Fälligkeitsdatum in 2 Tagen. Dort checkt Claude Code am Fälligkeitstag die Ads und verlinkt den zuständigen OMM mit einer Analyse.
8
Automatisch — Airtable
Stelle prüfen / anlegen
Prüft automatisch, ob die Stelle in Airtable bereits existiert. Falls ja → Status auf „Aktiv" setzen. Falls nein → neuen Datensatz anlegen und auf „Aktiv" setzen.
9
Automatisch — Trello
Kunde auf seinem Board benachrichtigen
Der Kunde bekommt automatisch eine Nachricht auf seinem persönlichen Trello-Board — vom Account des zuständigen OMM — dass die Kampagne erstellt wurde und in Kürze live geht.
10
Automatisch
Archivierung + Logging
Verwendete Creatives werden archiviert. Komplettes JSON-Log mit allen IDs, Einstellungen und Ergebnissen gespeichert.
!
Manuell — User muss prüfen
Bedingte Logik + Absage-Seite im Formular
Die Meta API unterstützt keine bedingte Logik. Deshalb muss der User nach der Erstellung manuell im Formular verknüpfen. Das kann nicht untergehen: Claude Code erinnert den User nach jeder Erstellung automatisch mit einer genauen Schritt-für-Schritt-Anleitung.
1. Bedingte Logik aktivieren
2. „Ja, ich bin bereit..." → weiter zu Frage 2
3. „Nein, möglichst günstig..." → Absage-Seite
4. Absage-Seite erstellen + IG-Profil verlinken
5. Erfolgsseite: IG-Profil + CTA „Mehr erfahren"
